사용자 입력기능
- 자동차 이름을 쉼표(,) 기준으로 구분하여 입력받는다.
- 5자를 초과한 이름이 입력되면 IllegalException을 발생시킨다.
- 시도할 횟수를 입력받는다.
- 0이상의 정수가 아닌 경우 IllegalException을 발생시킨다.

자동차 이동 기능
- 각 자동차에 0에서 9사이의 랜덤한 값을 생성한다.
- 무작위 값이 4이상인 경우만 전진한다.

경주 기능
- 각 시도마다 모든 자동차에 대해 "자동차이름 : 현재위치" 형식으로 위치를 출력한다.
- 가장 먼 거리를 간 최종 우승자를 결정해 "최종 우승자 : 자동차이름" 형식으로 출력한다.
- 우승자가 여려명이면 쉼표(,)로 이름을 구분해 출력한다.
